# side-by-side.dev

[![Netlify Status](https://api.netlify.com/api/v1/badges/0a307a49-98ca-4024-aff0-f8e304334de6/deploy-status)](https://app.netlify.com/projects/side-be-side-dev/deploys)

Comparing Frontend Frameworks.

## Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
